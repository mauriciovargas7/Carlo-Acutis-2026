# Pax vobiscum

## Milagres Eucarísticos

Este projeto é uma versão refatorada e responsiva de uma página sobre a exposição internacional **Os Milagres Eucarísticos no Mundo**, concebida e realizada por Carlo Acutis.

A proposta é organizar o conteúdo principal em uma interface mais clara, com navegação simples, leitura confortável e uma apresentação visual inspirada no site original da exposição.

> *Ad maiorem Dei gloriam.*
>
> Para a maior glória de Deus.

## Status do projeto

O projeto ainda está em fase inicial de desenvolvimento. Muitas abas, páginas internas e seções complementares ainda precisam ser criadas para que a experiência fique completa.

A formatação visual também será atualizada nas próximas etapas, buscando uma identidade mais alinhada ao padrão católico, com elementos gráficos, cores, tipografia e organização visual mais fiéis ao tema da exposição.

> *In principio erat Verbum.*
>
> No princípio era o Verbo.

## Recursos

- Cabeçalho com navegação para as seções principais.
- Carrossel visual com imagem de destaque.
- Link direto para o site original ao clicar na imagem.
- Blocos informativos sobre contato, leitura guiada e estatísticas da exposição.
- Estrutura preparada para expandir a lista de milagres e conteúdos relacionados.
